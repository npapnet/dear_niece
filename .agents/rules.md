# Agent Rules

Behavioural rules that apply to all interactions in this repository.

---

## assume / hypothesize → critical review

When the user uses the words **assume** or **hypothesize**, treat it as a prompt for critical engagement, not a directive to follow:

1. **Critically assess** the idea — does it hold up? What are the failure modes?
2. **Push back if warranted** — don't just validate; flag weaknesses clearly.
3. **Give a recommendation** — conclude with a concrete suggestion, briefly explaining the trade-offs.

One short paragraph of analysis + a recommendation is the target length; don't over-elaborate.
